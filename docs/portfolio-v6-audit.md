# Portfolio v6 visual and accessibility audit

The human-centered redesign replaces screenshot-like hero and flagship previews with semantic HTML evidence cards whose metrics, destinations and evidence boundaries remain readable at portfolio scale.

Final acceptance checks cover desktop, tablet and 390 px mobile rendering; light and dark themes; JavaScript-disabled output; horizontal overflow; generated HTML mirrors; four exact two-page application PDFs; the master CV PDF; link-label accessibility; color contrast; and production-like Lighthouse audits.

The Gray Swan live profile is the primary destination and the archive.is snapshot is retained only as a verification fallback. Wikimedia Commons links from the homepage point to a selected self-authored autism-genetics visualization rather than the complete upload history.
